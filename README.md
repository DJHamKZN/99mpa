# 99 МПа — сайт-презентация

Статический сайт архитектурного бюро «99 МПа»: обложка-монограф, кейсы и раздел
возможностей. Собран из дизайна Stitch/Figma и приведён к полностью автономному
виду — без внешних зависимостей (CDN, Google Fonts и т.п.).

## Страницы

| Файл | Раздел |
|------|--------|
| `index.html` | Обложка / архитектурный монограф |
| `projects.html` | Список проектов (хаб кейсов) |
| `bolgar.html` | Кейс Болгар: масштаб и детали |
| `bolgar-park.html` | Проект Болгар |
| `kamal-square.html` | Сквер у театра Камала |
| `umai-park.html` | Парк Умай |
| `approach.html` | Наш подход / о компании |
| `capabilities.html` | Возможности компании |
| `final.html` | Финальный разворот |

## Автономность

Все ресурсы лежат в репозитории — сайт работает без интернета к сторонним сервисам:

- **Стили** — Tailwind скомпилирован в статический `assets/css/styles.css` (без CDN-рантайма).
- **Шрифты** — Hanken Grotesk, JetBrains Mono и субсет Material Symbols self-hosted
  в `assets/fonts/` (`assets/css/fonts.css`), с кириллическими начертаниями.
- **Изображения** — все перенесены локально в `assets/img/`.

## Публикация (GitHub Pages)

Сайт статический, отдельная сборка не нужна. В настройках репозитория:
**Settings → Pages → Source: Deploy from a branch → `main` / `root`**.
Файл `.nojekyll` отключает обработку Jekyll, чтобы папка `assets/` отдавалась как есть.

Для домена `99mpa.ru` добавьте свой домен в **Settings → Pages → Custom domain**
и пропишите DNS согласно инструкции GitHub.
